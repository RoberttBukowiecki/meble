# Nano Banana Image Prompts for Blog Articles

This file contains the prompts for generating blog post images using the Nano Banana service.

## Pipeline Configuration
- **Target Directory:** `apps/landing2/public/img/blog/`
- **File Format:** `.webp`
- **Naming Convention:** `{slug}.webp`

## Articles & Prompts

### 1. Jak zaprojektować szafę na wymiar? (How to design a custom wardrobe?)
- **Slug:** `jak-zaprojektowac-szafe-na-wymiar-poradnik`
- **Filename:** `jak-zaprojektowac-szafe-na-wymiar-poradnik.webp`
- **Prompt:** `Modern interior design, photo realistic, a person measuring an empty wall alcove for a custom wardrobe, bright room, tape measure, professional tools, 4k, architectural visualization style`
